# LAB 3 - CRUD en React con Supabase

Aplicacion CRUD en React conectada a Supabase para administrar una lista de pendientes.

## Objetivo

Permitir consultar, agregar, actualizar y eliminar registros en la tabla `pendientes` de Supabase.

## Tecnologias

- React + Vite
- @supabase/supabase-js
- JavaScript (ES Modules)

## Estructura principal

- src/App.jsx: interfaz y logica CRUD
- src/supabase-client.js: configuracion de cliente Supabase
- src/App.css y src/index.css: estilos basicos

## Variables de entorno

Crea un archivo `.env` en la raiz con este contenido:

```env
VITE_SUPABASE_URL=https://TU-PROYECTO.supabase.co
VITE_SUPABASE_KEY=TU_SUPABASE_ANON_KEY
```

Nota: no compartas tu key real en repositorios publicos.

## Crear tabla en Supabase

En SQL Editor de Supabase, ejecuta:

```sql
create table if not exists public.pendientes (
	id bigint generated by default as identity primary key,
	name text not null,
	"isCompleted" boolean not null default false
);
```

Si usaras la clave anon desde frontend, habilita RLS y politicas para pruebas:

```sql
alter table public.pendientes enable row level security;

create policy "permite lectura anon"
on public.pendientes
for select
to anon
using (true);

create policy "permite insercion anon"
on public.pendientes
for insert
to anon
with check (true);

create policy "permite update anon"
on public.pendientes
for update
to anon
using (true)
with check (true);

create policy "permite delete anon"
on public.pendientes
for delete
to anon
using (true);
```

## Ejecutar el proyecto

1. Instala dependencias:

```bash
npm install
```

2. Ejecuta en desarrollo:

```bash
npm run dev
```

3. Compila para produccion:

```bash
npm run build
```

4. Revisa calidad de codigo:

```bash
npm run lint
```

## Funcionalidades implementadas

- Consulta registros al cargar con useEffect
- Agrega registros validando que no esten vacios
- Cambia estado de completado
- Elimina registros
- Refresca interfaz sin recargar pagina
- Manejo de errores con console.error
